# DISTRIBUTED TICKETING SYSTEM
============================

Project Overview
----------------
Distributed web-based ticketing platform designed to handle high-demand events,
secure user management, and concurrency-safe seat reservation workflows.

The system is architected as three independent subsystems:

    - esientradas   → Event and ticket management
    - esiusuarios   → Secure user account management
    - Front-end     → User interaction layer

The architecture focuses on modularity, scalability, and security best practices.
